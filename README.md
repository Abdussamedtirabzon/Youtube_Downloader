# 🚀 Advanced YouTube Downloader (MP3 & MP4)

Bu proje, Python ve Kivy kullanılarak geliştirilmiş, Linux ve Android platformlarında sorunsuz çalışan gelişmiş bir medya indirme aracıdır.

## ✨ Öne Çıkan Özellikler
* **Çoklu Format:** Videoları yüksek kalitede MP4 veya sadece ses olarak MP3 formatında indirebilirsiniz.
* **📦 Playlist Desteği:** Tekil videoların yanı sıra tüm YouTube playlistlerini otomatik olarak algılar ve toplu şekilde indirir.
* **🐧 Linux Optimizasyonu:** Fedora başta olmak üzere tüm modern Linux dağıtımlarında terminal üzerinden veya GUI ile çalışabilir.
* **📱 Android Hazır:** Buildozer entegrasyonu sayesinde mobil uygulama (APK) olarak paketlenmeye uygundur.
